# Python Collections Assignment

## Overview

This project demonstrates the use of Python's core collection types:

-   Lists
-   Tuples
-   Sets
-   Dictionaries

It also combines these data structures to build a simple product
catalog.

## Tasks

### Task 1: Product Collections

-   Create a list of products.
-   Create a tuple containing product details.
-   Print the second and last products.
-   Append new products.
-   Convert a tuple to a list, update a value, and convert it back.
-   File Name : Assignment1_Task1.py

### Task 2: Categories (Sets)

-   Create a set of product categories.
-   Demonstrate that duplicate values are ignored.
-   Check if a category exists.
-   Count the number of unique categories.
-   File Name : Assignment1_Task2.py

### Task 3: Product Pricing (Dictionaries)

-   Create a dictionary of product prices.
-   Add, update, and remove products.
-   Calculate the average price.
-   Find the highest and lowest priced products.
-   File Name : Assignment1_Task3.py

### Task 4: Combined Operations

-   Create a catalog as a list of tuples.
-   Build a dictionary mapping categories to products.
-   Find the category containing the maximum number of products.
-   File Name : Assignment1_Task4.py
